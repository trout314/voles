# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) code of conduct. In brief: be respectful, constructive, and welcoming to contributors of all backgrounds. Issues and pull requests that are disrespectful or harassing will be closed.

To report a concern, open a GitHub issue marked **[conduct]** or contact the maintainer directly via GitHub.
